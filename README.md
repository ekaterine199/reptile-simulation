# рЃЦрЃЋрЃћрЃгрЃљрЃарЃЏрЃљрЃЋрЃџрЃћрЃЉрЃўрЃА рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃўрЃА рЃЏрЃљрЃарЃћрЃњрЃБрЃџрЃўрЃарЃћрЃЉрЃћрЃџрЃў рЃЊрЃљ рЃЏрЃљрЃЏрЃЮрЃФрЃарЃљрЃЋрЃћрЃЉрЃћрЃџрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ

рЃћрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА рЃЦрЃЋрЃћрЃгрЃљрЃарЃЏрЃљрЃЋрЃџрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃФрЃарЃљрЃЮрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЌрЃћрЃарЃЏрЃЮрЃарЃћрЃњрЃБрЃџрЃљрЃфрЃўрЃўрЃА рЃАрЃўрЃЏрЃБрЃџрЃљрЃбрЃЮрЃарЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃюрЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџрЃўрЃљ рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃЏрЃўрЃќрЃюрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃАрЃарЃБрЃџрЃљрЃЊ рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃўрЃљ рЃћрЃарЃЌ HTML рЃцрЃљрЃўрЃџрЃерЃў рЃЊрЃљ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃбрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ, рЃЏрЃљрЃЌ рЃерЃЮрЃарЃўрЃА рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃа рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃќрЃћ.

## ­Ъдј рЃАрЃљрЃЏрЃўрЃќрЃюрЃћ рЃфрЃюрЃћрЃЉрЃћрЃЉрЃў

рЃћрЃА рЃАрЃўрЃЏрЃБрЃџрЃљрЃфрЃўрЃљ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃўрЃФрЃџрЃћрЃЋрЃљ, рЃарЃЮрЃЏ рЃЊрЃћрЃЏрЃЮрЃюрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃўрЃДрЃЮрЃА рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЉрЃўрЃЮрЃџрЃЮрЃњрЃўрЃБрЃарЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў:

- рЃЦрЃЋрЃћрЃгрЃљрЃарЃЏрЃљрЃЋрЃџрЃћрЃЉрЃўрЃА рЃАрЃ«рЃћрЃБрЃџрЃўрЃА рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃљ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃљрЃќрЃћ
- рЃ«рЃЋрЃџрЃўрЃЎрЃћрЃЉрЃў рЃњрЃљрЃЊрЃљрЃљрЃЊрЃњрЃўрЃџрЃЊрЃћрЃЉрЃўрЃљрЃю рЃгрЃўрЃюрЃљ рЃЊрЃљ рЃБрЃЎрЃљрЃюрЃљ рЃЎрЃўрЃЊрЃБрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ
- рЃњрЃЋрЃћрЃџрЃћрЃЉрЃў рЃњрЃљрЃЊрЃљрЃљрЃЊрЃњрЃўрЃџрЃЊрЃћрЃЉрЃўрЃљрЃю рЃбрЃљрЃџрЃдрЃўрЃАрЃћрЃЉрЃарЃў рЃАрЃ«рЃћрЃБрЃџрЃўрЃА рЃЏрЃЮрЃФрЃарЃљрЃЮрЃЉрЃўрЃЌ рЃЊрЃљ рЃЎрЃўрЃЊрЃБрЃарЃћрЃЉрЃў рЃљрЃа рЃњрЃљрЃљрЃЕрЃюрЃўрЃљрЃЌ

## РџЎ№ИЈ рЃЏрЃљрЃарЃЌрЃЋрЃљрЃЊрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў

- **рЃ«рЃћрЃЊрЃў**: рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃљрЃўрЃарЃЕрЃўрЃЮрЃА рЃ«рЃЋрЃџрЃўрЃЎрЃў рЃљрЃю рЃњрЃЋрЃћрЃџрЃў
- **рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃљ**: 0┬░C-рЃЊрЃљрЃю 45┬░C-рЃЏрЃЊрЃћ  
  Рєњ рЃњрЃљрЃЋрЃџрЃћрЃюрЃљрЃА рЃљрЃ«рЃЊрЃћрЃюрЃА рЃАрЃ«рЃћрЃБрЃџрЃўрЃА рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃљрЃќрЃћ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃюрЃћрЃџрЃљ рЃћрЃарЃњрЃћрЃЉрЃљ рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљрЃА (~5-10 рЃгрЃљрЃЏрЃўрЃА рЃЊрЃљрЃњрЃЋрЃўрЃљрЃюрЃћрЃЉрЃўрЃЌ)
- **рЃЏрЃЮрЃФрЃарЃљрЃЮрЃЉрЃўрЃА рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ/рЃњрЃљрЃЕрЃћрЃарЃћрЃЉрЃљ**: рЃЦрЃЋрЃћрЃгрЃљрЃарЃЏрЃљрЃЋрЃџрЃўрЃА рЃЏрЃЮрЃФрЃарЃљрЃЮрЃЉрЃўрЃА рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃў рЃЕрЃљрЃарЃЌрЃЋрЃљ/рЃњрЃљрЃЏрЃЮрЃарЃЌрЃЋрЃљ
- **рЃЎрЃўрЃЊрЃБрЃарЃћрЃЉрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ** (рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃ«рЃЋрЃџрЃўрЃЎрЃўрЃАрЃЌрЃЋрЃўрЃА): рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃцрЃћрЃ«рЃћрЃЉрЃўрЃА рЃЕрЃљрЃарЃЌрЃЋрЃљ/рЃњрЃљрЃЏрЃЮрЃарЃЌрЃЋрЃљ
- **рЃбрЃљрЃџрЃдрЃўрЃА рЃАрЃўрЃЕрЃЦрЃљрЃарЃћ** (рЃњрЃЋрЃћрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА): рЃАрЃ«рЃћрЃБрЃџрЃўрЃА рЃбрЃљрЃџрЃдрЃўрЃАрЃћрЃЉрЃарЃў рЃЏрЃЮрЃФрЃарЃљрЃЮрЃЉрЃўрЃА рЃАрЃўрЃЕрЃЦрЃљрЃарЃћ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ рЃАрЃ«рЃћрЃБрЃџрЃўрЃА рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃљрЃќрЃћ

## ­Ъњ╗ рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў

- ­ЪћЂ рЃАрЃарЃБрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃўрЃарЃћрЃЉрЃљ рЃћрЃарЃЌ HTML рЃцрЃљрЃўрЃџрЃерЃў
- ­Ъїљ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃбрЃўрЃА рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ
- ­ЪЊ▒ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃћрЃарЃњрЃћрЃЉрЃљ рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃа рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃА (`responsive`)
- ­Ъје рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў рЃАрЃарЃБрЃџрЃљрЃЊ рЃЦрЃљрЃарЃЌрЃБрЃџ рЃћрЃюрЃљрЃќрЃћ

## ­Ъје рЃЉрЃарЃћрЃюрЃЊрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃЊрЃўрЃќрЃљрЃўрЃюрЃў

- UI рЃћрЃюрЃљ: **рЃЦрЃљрЃарЃЌрЃБрЃџрЃў**
- рЃЎрЃЮрЃЏрЃърЃљрЃюрЃўрЃўрЃА рЃџрЃЮрЃњрЃЮ рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃЏрЃљрЃарЃфрЃ«рЃћрЃюрЃљ рЃќрЃћрЃЊрЃљ рЃЎрЃБрЃЌрЃ«рЃћрЃерЃў
- рЃцрЃћрЃарЃћрЃЉрЃўрЃА рЃърЃљрЃџрЃћрЃбрЃў: **рЃерЃљрЃЋрЃў**, **рЃЌрЃћрЃЌрЃарЃў**, **рЃџрЃБрЃарЃ»рЃў**
